How It Works

Start daemon on your machine — ud daemon
↓
2
Open a task in the web UI and click the workspace icon
↓
3
Select a daemon — the server pushes instructions via SSE
↓
4
Agent runs — reads the task, executes, writes progress notes
↓
5
Watch live — notes stream into the task page in real-time

🚫

🔑
API key auth — daemon requires valid credentials to connect
⚙️
Custom config — define your own rules in workspace-blacklist.yml
