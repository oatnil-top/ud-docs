Level 1 — Board Scope

  • The implicit access-control layer — no configuration needed
  • Private boards: only you see your own tasks
  • Shared boards: team members in the same group see shared tasks
  • This is the broadest filter — it defines who can see what

Level 2 — Scope Tags

  • Configured in Board Settings → Scope Tags
  • Only tasks with all specified scope tags appear on the board
  • New tasks created on the board automatically receive these tags
  • Example: Board "Sprint 1" with scope tag sprint-1 shows only sprint-1 tasks
All Tasks (40+) → sprint-1 tagged (8)

Level 3 — Column Query

  • Each column defines its own match condition
  • Can combine multiple conditions per column
  • Drag-and-drop auto-applies the target column's condition

Level 4 — Ephemeral Board Filter

  • Real-time filtering via search bar, tag badges, and assignee filter
  • Persisted in URL: ?tags=backend&search=API
  • Shareable — send the URL to a teammate
  • Temporary — does not modify the board configuration
